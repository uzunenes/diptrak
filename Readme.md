# dvmot
Deep Visual Multiple Object Tracking


## Prerequisites
* [darknet](https://github.com/AlexeyAB/darknet)
* [opencv3.4](https://github.com/opencv/opencv/tree/3.4)


## Usage
Compile and run dvmot
```
$ make
$ ./dvmot.out dvmot.ini
```

## Pre-trained models
